# SDG 12.3 Food Waste Analytics Dashboard – Ireland

## Overview
This project analyses Ireland’s progress towards the United Nations Sustainable Development Goal 12.3, which focuses on reducing food waste across the supply chain. The solution combines Python-based data preparation with an interactive Power BI dashboard to generate actionable insights on environmental and social impact.

## Business Objective
The objective of this project is to:
- Monitor food waste trends across different sectors
- Evaluate progress towards the 2030 reduction target
- Quantify the environmental impact in terms of carbon emissions
- Highlight the social implications of food waste
- Support data-driven decision-making for stakeholders

## Key Insights
- Ireland is currently not on track to meet the 2030 food waste reduction target
- Commercial sectors such as processing, retail, and hospitality contribute the largest share of food waste
- Food waste has a direct impact on greenhouse gas emissions
- A significant portion of wasted food could be redirected to address food insecurity
- Waste management practices are improving, but further optimization is required

## Tools and Technologies
- Python for data cleaning and preprocessing
- Power BI for dashboard development and data visualization
- Pandas and NumPy for data manipulation
- Public datasets from EPA, FAO, and EEA

## Data Pipeline
The project follows a structured data pipeline:
Raw Data → Data Cleaning (Python) → Feature Engineering → Power BI (DAX) → Dashboard Visualization

## Project Structure
- data: raw and processed datasets
- notebooks: Python notebooks for data preparation
- dashboard: Power BI dashboard file
- report: final project report
- images: dashboard screenshots

## Key Features
- KPI-based tracking of baseline, current, and target values
- Time-series analysis of food waste trends
- Derived metrics such as carbon emissions and emission intensity
- Social impact metric estimating potential population fed
- Multi-level analysis across sectors and waste management systems

## Skills Demonstrated
- Data cleaning and preprocessing
- Handling missing data and outliers
- Feature engineering and data transformation
- Data modeling and visualization
- Dashboard design and storytelling
- Business insight generation

## How to Use
1. Open the Power BI file located in the dashboard folder
2. Run the notebook in the notebooks folder to reproduce data preparation steps
3. Install dependencies using:
   pip install -r requirements.txt
